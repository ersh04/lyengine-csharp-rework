# lyengine-csharp-rework

`lyengine-csharp-rework` is a repository for rebuilding **LyEngine** in **C#**.

## Status

This project is currently at an early bootstrap stage. The repository is being prepared for the C# rework, so source code and setup instructions will be expanded as implementation begins.

## Goals

- Rework LyEngine in C#
- Establish a clean repository structure for the new implementation
- Document setup, build, and usage instructions as the codebase grows

## Getting Started

At the moment, there is no application code or solution file in the main repository brench yet.
